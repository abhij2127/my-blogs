    Data Driven Documents with D3.js

    What if someone told you that there is a way to add dynamic contents to your webpage without the tedious task of adding vannila javascript. "Well", you may say, "Cerytainly I can use react or any other modern component based frameworks", not that simple. The overhead of installing such large libraries just to add some dynamic content, moreover what if the data that you are recieving has to be shown in a Barchart or Scatter Chart, again you might say that there are awesome libraries like chart.js and google charts which I can use with frameworks, yes but again the overhead is too much. D3.js is a library that solves all of the problem by providing an easy to use interface and syntactical sugar. Let's see how.

    So now you have made up mind to add a new library to your resume and you are thinking where to start. It can be quite overwhelming with all the resources available on the internet, don't worry a list of best resources to refer to is compiled for you and you can check it here.

    It is so simple to get started with using D3.js. Just add below line in your html page.

    <script src="https://d3js.org/d3.v7.min.js"></script>

    So now you have added D3.js successfully. And we are all set to use it, but how?
    " Note - If you want to use D3.js with frameworks like react, angular etc you can install it using npm. See official documentation here for more info".
